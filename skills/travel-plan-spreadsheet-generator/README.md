# travel-plan-spreadsheet-generator

Build a polished, editable `.xlsx` travel itinerary workbook from messy travel notes, PDFs, screenshots, shopping asks, and fixed commitments.

This skill is for spreadsheet output, not prose-only itinerary writing. It keeps bookings, daily plans, prep/compliance, pack list, buy list, and sources in one calm workbook.

Example prompts:

- "Turn these flight PDFs, hotel screenshots, and shopping notes into a travel itinerary spreadsheet."
- "Make me an `.xlsx` conference travel workbook with daily plans, prep tasks, and a buy list."
- "I have messy holiday notes and partial bookings. Build a trip spreadsheet and flag what still needs review."

The generated artifact is an `.xlsx` workbook.
