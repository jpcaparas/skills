# Research Policy

Verify volatile travel facts when they materially affect the workbook.

## When Research Is Mandatory

For current or future trips, verify these when relevant:

- visa or entry rules by passport nationality
- transit or transfer requirements
- official customs or immigration tooling
- airline check-in windows
- baggage or dangerous-goods basics when the user mentioned luggage, medicines, batteries, or powerbanks
- travel-health guidance when vaccines or health concerns were raised
- medicine import rules when medicines are mentioned
- conference or event dates when those dates drive the plan

## Source Priority

Use primary and official sources first:

- embassy, immigration, border, customs, and government travel portals
- airline help centers and fare-rule pages
- official event or conference pages
- official venue or ticketing pages
- destination or health authorities

Use secondary sources only to point you toward an official page, not as the final authority.

## Scope Control

- Only add prep tasks that actually matter for this route, these travellers, and these notes.
- Do not pad the workbook with generic health scares or filler admin.
- When a risk is low-impact or already clearly covered, note it briefly or omit it.

## Recording Sources

Every source actually used must land in `Sources` with:

- source type
- source name
- plain-text URL or file note
- what it was used for
- last checked or file date
- a concise practical note

## Contradictions

Do not silently overwrite one source with another.

Examples:

- hotel says 3 adults, planner assumes 4
- outbound flight covers only 2 travellers
- attractions cover only part of the group
- return itinerary visibly names one traveller only
- pasted notes reference weekdays from a different calendar year

These become review flags, not silent normalizations.

## Time Sensitivity

If the harness provides `{{ skill:temporal-awareness }}`, load it or apply the same logic before researching volatile facts. Use exact dates in your notes, not only relative phrasing such as "today" or "this week".
