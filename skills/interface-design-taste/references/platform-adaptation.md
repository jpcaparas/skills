# Platform Adaptation

The same visual language should not behave the same way everywhere.

## Web Marketing

Primary user mode:

- browsing
- comparing
- building confidence

What matters most:

- narrative pacing
- clear primary call to action
- memorable hierarchy
- a strong first impression

Design moves that usually help:

- bigger type contrasts
- bolder image treatment
- section-level pacing changes
- more expressive motion in hero or transition zones

What usually hurts:

- desktop-app density
- overcomplicated navigation
- too many equally weighted cards

## Web Application

Primary user mode:

- completing tasks
- checking status
- moving between repeated flows

What matters most:

- reliable navigation
- fast scanability
- predictable forms and filters
- clear state feedback

Design moves that usually help:

- calmer visual system
- stronger repeated component rules
- tighter control over accents and motion
- better empty, loading, and error states

What usually hurts:

- launch-page hero logic inside core product areas
- excessive asymmetry in repeated work surfaces
- motion that delays task completion

## Desktop Application

Primary user mode:

- longer sessions
- denser work
- higher precision
- frequent context switching

What matters most:

- window and pane behavior
- tool discoverability
- keyboard support
- secondary and tertiary controls
- strong information density without clutter

Design moves that usually help:

- split panes, inspectors, and utility rails
- denser but regular spacing
- hover, selection, and focus states that support precise work
- restrained depth and fast transitions

What usually hurts:

- giant marketing-style typography
- oversized cards for every object
- soft luxury treatments that obscure structure
- animations that disrupt focus during repeated tasks

## Cross-Platform Translation

When the same product spans web and desktop:

- keep the core type and color logic consistent
- let density, chrome, and interaction depth diverge
- do not force identical navigation if the environment changes
- let desktop expose more controls while web stays more guided

## Quick Mapping

| Constraint | Lean Toward |
| --- | --- |
| heavy repeated work | Product Precision or Operational Density |
| content trust and reading | Editorial Calm |
| premium consumer feel | Warm Material |
| campaign or reveal page | Expressive Showcase |
