# Interface Design Taste

Portable production skill for shaping web, app, and desktop interfaces with stronger hierarchy, clearer visual systems, and more product-appropriate taste.

The authoritative instructions live in `SKILL.md`.

## Key Files

- `SKILL.md` for the canonical design router
- `references/principles.md` for core taste rules and planning axes
- `references/style-families.md` for coherent visual lanes
- `references/layout-and-rhythm.md` for hierarchy and composition
- `references/platform-adaptation.md` for web, app, and desktop differences
- `references/critique-workflow.md` for redesign-first audits
- `references/design-systems-and-tokens.md` for reusable system capture
