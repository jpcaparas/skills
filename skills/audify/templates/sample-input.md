# Audify Sample Input

This sample exists so the skill can ship real, runnable file examples.

Visit [the Gemini speech generation guide](https://ai.google.dev/gemini-api/docs/speech-generation) if you want the official docs later, but the spoken transcript should keep only the visible anchor text.

Bare URLs like https://example.com/should-not-be-read-aloud should disappear from the cleaned narration.

The output should sound like a short, natural explainer rather than a parser reading transport syntax.
