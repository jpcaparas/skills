# markdown.new

Production skill for using markdown.new's URL conversion, file conversion, crawl, and editor surfaces.

## What It Covers

- Single-page URL-to-Markdown flows
- Remote file and local upload conversion
- Crawl job lifecycle and caching behavior
- Hosted UI pages and editor utilities
- Verified edge cases from live testing on April 9, 2026

## Key Files

- `SKILL.md` - authoritative instructions
- `references/url-conversion.md` - one-page conversion patterns
- `references/file-conversion.md` - remote file and local upload flows
- `references/crawl.md` - crawl API and status handling
- `references/gotchas.md` - failure cases and quirks
- `scripts/probe_markdown_new.py` - repeatable HTTP probe helper
