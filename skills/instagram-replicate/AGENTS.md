# instagram-replicate

Use `SKILL.md` as the source of truth for this skill.

This package exists so public skill installers can discover a deterministic Instagram replication workflow that:

- freezes one public post into `snapshot.json`
- keeps a rerenderable local build folder
- exports both `instagram-replica.mp4` and `instagram-replica.gif`
- enforces a GIF size cap of 24 MB
