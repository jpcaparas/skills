# eli12 References

Load only what the current explanation needs.

## File guide

| Need | Read |
| --- | --- |
| Explore a broad subsystem before explaining it | `references/explorer-prompt.md` |
| Write the human-facing explanation | `references/explainer-prompt.md` |
| Choose or debug analogies | `references/analogy-patterns.md` |
| Recover from misleading simplifications | `references/gotchas.md` |

## Operating principle

Gather evidence first, simplify second.

The explorer prompt is for code reading and evidence collection. The explainer prompt is for turning that evidence into a clear mental model. The analogy guide helps only after the real structure is understood.
